The result set contains the following data:

"Subject" - Subject id, 1- 30
"Activity" - Activity the SUbject was performing. Can be "SITTING", "LAYING", "WALKING", "WALKING DOWNSTAIRS", "WALKING UPSTAIRS", "STANDING"

Following columns are extracted from the training data set. The prefix of column name signifies the measurement type. If the name contains mean() - the column contains mean value of all values of the measurement for the subject performing the activity. Same goes for std() (mean of standard deviation)
